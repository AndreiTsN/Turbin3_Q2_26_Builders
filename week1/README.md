# Week 1 — Guessing Game

## Overview
A simple CLI game written in Rust based on the official Rust book.

The program generates a random number and asks the user to guess it.

## What was done
- Created a Rust project using Cargo
- Implemented user input handling
- Generated a random number
- Added comparison logic (too small / too big / correct)
- Handled basic errors

cd guessing_game_rust_book
